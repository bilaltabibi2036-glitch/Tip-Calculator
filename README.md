# 💰 Tip Calculator

> A clean and simple tip calculator built with Python.

---

## 🚀 About

This project was built on **Day 2** of the **100 Days of Code** challenge.
The goal was to practice user input, variables, and basic arithmetic in Python.

---

## ✨ Features

- ✅ Takes the total bill as input
- ✅ Lets you choose a tip percentage (10%, 12%, or 15%)
- ✅ Splits the bill between any number of people
- ✅ Rounds the final amount to 2 decimal places

---

## 🛠️ Built With

- Python 3

---

## ▶️ How to Run

```bash
python main.py
```
